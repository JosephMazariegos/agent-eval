# agent-eval
Platform for evaluating AI Coding assistants using metrics and telemetry

## Features (planned)
- Task creation and management
- AI-genereated code submission tracking
- Automated evaluation (tests, performance, linting)
- Metrics dashboard comparing tools
- Prompt optimization experiments

## Stack
- Backend: FastAPI (Python)
- Frontend: React
- Database: SQLite (initially)

## Status
Currently in early stage development
